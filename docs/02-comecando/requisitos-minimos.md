# Requisitos mínimos

Antes de iniciar as atividades do programa, é importante verificar se você possui as condições mínimas para acompanhar o treinamento e realizar as etapas práticas com tranquilidade.

Esta página apresenta uma lista básica de requisitos técnicos e operacionais. Dependendo da turma, da ferramenta utilizada e da forma de condução das atividades, pode haver orientações complementares.

## Requisitos básicos

Para participar adequadamente, é necessário ter:

- 💻 um computador com acesso à internet, sistema operacional *Windows* e permissão para instalação de programas.

## Quando pedir ajuda

Procure apoio da equipe de treinamento se:

- algum site não abrir;
- a ferramenta principal não carregar;
- houver erro de acesso ou permissão;
- você não souber qual ambiente deve usar;
- seu computador estiver muito limitado para acompanhar as etapas.

## Próxima leitura recomendada

Depois desta verificação, siga para **Primeiros passos** para começar sua preparação inicial no programa.
