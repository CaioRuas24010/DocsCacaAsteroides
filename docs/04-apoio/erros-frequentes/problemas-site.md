# Problemas relacionados ao site do IASC

Aqui, encontrará uma lista de problemas relacionados ao site do IASC, que é a plataforma onde as imagens são disponibilizadas e onde os relatórios são enviados. Se você estiver enfrentando algum problema relacionado ao acesso às imagens, envio de relatórios ou qualquer outra questão relacionada ao site, confira as soluções abaixo.

## Erro ao acessar página da equipe

Se ao tentar acessar a página da sua equipe no site do IASC o navegador apenas retornar a página inicial do site, isso pode indicar um problema de acesso à conta.

- Verifique se está conectado com a conta correta.
- Verifique se está clicando na equipe correta, ou seja, a equipe correspondente ao número informado por email.

Se estiver com mais dúvidas, verifique a seção [Abrindo imagens](../../03-tutorial-pratico/abrir-imagens.md) do tutorial prático, onde há um passo a passo detalhado de como acessar as imagens e a página da equipe no site do IASC. Caso o problema persista, entre em contato com a equipe de suporte para obter assistência.

## Erro ao enviar relatório

O erro mais comum ao enviar o relatório é uma mensagem indicando erro no código F51. Isso geralmente indica um problema com o formato do relatório, entretanto, é possível que ele apareça mesmo que o formato do relatório esteja correto. Nesse caso, tente a seguinte solução:

- Tente enviar o relatório utilizando um navegador diferente. O site do IASC é otimizado para funcionar melhor no Google Chrome e pode apresentar problemas em outros navegadores.
- Tente utilizar outra conexão de internet, como uma rede de celular, para descartar problemas relacionados à rede.

Caso seja um erro diferente, verifique se o formato do relatório está correto, seguindo as orientações descritas na seção [Enviando relatório](../../03-tutorial-pratico/enviar-relatorio.md) do tutorial prático. Se o problema persistir, entre em contato com a equipe de suporte para obter assistência.
