# Documentação do Programa Caça Asteroides MCTI

Bem-vindo à documentação de apoio para participantes do Programa Caça Asteroides MCTI no Brasil.

Este site foi criado para funcionar como um guia de consulta rápida durante a formação e durante as atividades práticas. Em vez de depender de materiais estáticos e difíceis de atualizar, esta documentação reúne orientações, procedimentos e respostas para dúvidas frequentes em um só lugar.

<span style={{backgroundColor: '#b4f9f7', padding: '0.1rem 0.25rem'}}>🔎 É possível utilizar a pesquisa no canto superior direito para buscar um assunto específico, como um erro que você está enfrentando ou uma etapa do processo. 🔍</span>

## O que você vai encontrar aqui

Nesta documentação, você encontrará:

- uma visão geral do programa;
- tutoriais práticos de consulta;
- como corrigir erros comuns;

## Como usar esta documentação

Se você está começando agora, siga esta ordem:

1. Verifique os requisitos mínimos.
2. Siga a página de primeiros passos.
3. Avance para o tutorial prático.

Se você já participou de atividades anteriores, use o menu lateral para ir diretamente ao assunto que deseja revisar.

## Objetivo desta documentação

O objetivo deste site é facilitar o aprendizado, reduzir dúvidas repetidas e manter os procedimentos organizados de forma clara e atualizável.

Ao longo do tempo, novas páginas, exemplos e orientações poderão ser adicionados conforme as necessidades do programa.

## Caminho recomendado

Para começar, avance para **Começando**.
