# MCTI Asteroid Hunt Program Documentation

Welcome to the support documentation for participants in the MCTI Asteroid Hunt Program in Brazil.

This site was created to serve as a quick reference guide during training and practical activities. Instead of relying on static materials that are difficult to update, this documentation brings together guidance, procedures, and answers to frequently asked questions in one place.

<span style={{backgroundColor: '#acb804', padding: '0.1rem 0.25rem'}}>🔎 You can use the search bar in the top right corner to look up a specific topic, such as an error you are facing or a step in the process. 🔍</span>

## What you will find here

In this documentation, you will find:

- an overview of the program;
- practical reference tutorials;
- how to fix common errors;

## How to use this documentation

If you are just getting started, follow this order:

1. Check the minimum requirements.
2. Follow the getting started page.
3. Move on to the practical tutorial.

If you have already participated in previous activities, use the side menu to go directly to the topic you want to review.

## Purpose of this documentation

The purpose of this site is to make learning easier, reduce repeated questions, and keep procedures organized in a clear and updatable way.

Over time, new pages, examples, and guidance may be added as the program evolves.

Please, if you have any suggestions for improving the documentation, feel free to share them with the training team. Your feedback is very valuable to make this resource as useful as possible for everyone. Also, english is not our first language, so if you find any errors or confusing parts in the text, please let us know so we can correct them.

## Recommended path

To begin, move on to **Getting Started**.
