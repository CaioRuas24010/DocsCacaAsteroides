# Problems related to the IASC website

Here you will find a list of problems related to the IASC website, which is the platform where images are made available and where reports are submitted. If you are facing any issue related to accessing images, submitting reports, or any other website-related matter, check the solutions below.

## Error when accessing the team page

If, when trying to access your team page on the IASC website, the browser simply returns to the website’s home page, this may indicate an account access problem.

- Make sure you are logged in with the correct account.
- Make sure you are clicking on the correct team, that is, the team corresponding to the number sent by email.

If you still have questions, check the [Opening images](../../03-tutorial-pratico/abrir-imagens.md) section of the practical tutorial, where there is a detailed step-by-step explanation of how to access the images and the team page on the IASC website. If the problem persists, contact the support team for assistance.

## Error when submitting the report

The most common error when submitting the report is a message indicating an error in code F51. This usually indicates a problem with the report format; however, it may also appear even when the report format is correct. In that case, try the following solutions:

- Try submitting the report using a different browser. The IASC website is optimized to work best in Google Chrome and may present problems in other browsers.
- Try using a different internet connection, such as a mobile network, to rule out network-related issues.

If it is a different error, check whether the report format is correct by following the instructions described in the [Submitting report](../../03-tutorial-pratico/enviar-relatorio.md) section of the practical tutorial. If the problem persists, contact the support team for assistance.
